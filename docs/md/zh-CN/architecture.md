## 组件

### 接口

HMG 提供四个访问面：

| 接口 | 协议 | 使用场景 |
|---|---|---|
| **MCP** | Model Context Protocol | Agent 集成（主要方式） |
| **HTTP API** | REST + JSON | SDK 集成、自定义工具 |
| **CLI** | 终端（`hmg` 命令） | 管理、调试、脚本 |
| **TUI** | 交互式终端（`hmg tui`） | 可视化浏览和管理 |

四个面暴露相同的能力——存储记忆、召回、纠错和治理。

### 记忆引擎

HMG 的核心。管理以下内容：

- **图谱模型**：通过类型化边连接的原子（Supersedes、DerivesFrom、RelatesTo 等）
- **索引**：关键词搜索、时序排序、分类分组，以及（Developer+ 版本中的）语义搜索
- **存储**：带快照历史的持久化本地存储

### Agent 集成

`hmg init --agent <id>` 配置 agent 使用 HMG 作为其记忆层。支持的 agent：

| Agent | 状态 |
|---|---|
| Cursor | ✅ 已支持 |
| pi (Codex fork) | ✅ 已支持 |

| Codex | ✅ 已支持 |

| Aider | ✅ 已支持 |
| Continue | ✅ 已支持 |

## 数据流

### 记忆

```
Agent → "记住这个：..." → HMG
  → 验证输入
  → 创建类型化记忆原子
  → 附加作用域和上下文
  → 建立检索索引（关键词 + 时序 + 分类）
  → 持久化到本地存储
  → 返回原子 ID + 确认
```


### 召回

```
Agent → "数据库用什么？" → HMG
  → 解析查询意图
  → 从索引检索候选
  → 按相关性、确定性、时效排序
  → 按作用域和治理状态过滤
  → 通过图遍历投影相关原子
  → 按 Agent Tool Output Contract 格式化输出
  → 返回结构化结果 + 诊断信息
```


### 纠错

```
Agent → "那个不对，实际上是 ..." → HMG
  → 创建纠错原子（动作 + 原因）
  → 通过 Supersedes 边链接
  → 更新原始原子的极性/认识状态
  → 持久化纠错历史
  → 返回纠错确认
```


### 治理

```
管理员 → "密封这条敏感记忆" → HMG
  → 验证治理动作
  → 转换暴露状态（可见 → sealed）
  → 可选地派生安全教训原子
  → 持久化治理记录
  → 原始内容变为不可检索（sealed）
```


## 存储

HMG 将所有数据存储在运行机器的本地：

```
~/.local/share/hmg/
  stores/
    default/           ← 默认记忆存储
      graph/           ← 原子和边数据
      indexes/         ← 搜索索引
      snapshots/       ← 纠错/治理历史
```

Community 和 Developer Local 版本中，数据不会离开本机。

## Edition 架构

HMG 是一个包含所有版本代码的单一二进制。活动版本在启动时确定：

```
HMG 二进制
  │
  ├── 无 license key → Community Edition
  │     └── 关键词搜索，50K 原子，5 个 agent，基础功能
  │
  ├── HMG_LICENSE_KEY=<your-key> → Developer Edition
  │     └── One-Shot Recall，自动整理，Domain Packs，无限量
  │
  ├── HMG_LICENSE_KEY=<your-key> → Enterprise Edition
  │     └── 全部功能，SSO，RBAC，多租户，审计
  │
  └── HMG_CLOUD_TOKEN → 云连接
        └── 通过云认证获得 Developer 或 Enterprise
```

这意味着：
- 不需要维护多个二进制
- 升级即时生效：`export HMG_LICENSE_KEY=...` 然后重启
- Community 用户获得与 Enterprise 相同的二进制质量

## 安全边界

```
┌─────────────────────────────────┐
│         Agent 进程               │
│   （以用户权限运行）               │
└──────────────┬──────────────────┘
               │ MCP / HTTP (localhost)
               ▼
┌─────────────────────────────────┐
│         HMG 进程                 │
│   （绑定到 localhost:3000）       │
│                                  │
│   记忆数据：仅用户可访问           │
│   无出站连接 (CE)                 │
│   无遥测 (CE)                    │
└─────────────────────────────────┘
```

- Community Edition **零出站网络连接**
- HMG 默认绑定到 `localhost`——不暴露到网络
- 存储文件使用仅用户权限
