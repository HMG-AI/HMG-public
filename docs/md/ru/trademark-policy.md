# Политика товарных знаков HMG

**Последнее обновление:** 25 мая 2026 г.

## Товарные знаки HMG
Товарные знаки 武汉凡尘合创科技有限公司: **HMG**, **Holographic Memory Graph**, **HMG Cloud**, **HMG Developer**, **HMG Enterprise**, **HMG Compatible**, **HMG Certified**, **One-Shot Recall**, логотип HMG.

*(Примечание: Регистрация товарных знаков в процессе.)*

## Руководящие принципы
### Разрешено
- Использовать «HMG» в обычном тексте для ссылки на продукт
- Заявлять «реализует HMG Protocol» или «совместим с HMG»
- Использовать в документации, блогах, образовательных материалах

### Запрещено
- Использовать товарные знаки HMG без письменного разрешения в названии продукта, домене или логотипе
- Подразумевать официальную поддержку без авторизации

### «HMG Compatible»: Только после прохождения `hmg-certification`
### «HMG Certified»: Только после официального процесса сертификации
