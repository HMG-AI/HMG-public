# Архитектура HMG

## TUI
```bash
hmg tui
```


## Обзор системы
```
AI Agent / IDE → (MCP / HTTP) → Бинарный файл HMG → Движок памяти → Локальная файловая система
```

## Интерфейсы
| Интерфейс | Протокол | Использование |
|---|---|---|
| **MCP** | Model Context Protocol | Интеграция агента (основная) |
| **HTTP API** | REST + JSON | Интеграция SDK |
| **CLI** | Терминал (`hmg`) | Администрирование |
| **TUI** | Интерактивный терминал (`hmg tui`) | Визуальный просмотр |

## Поддерживаемые агенты
Cursor, pi, Claude Code, Codex, Windsurf, Aider, Continue — все ✅

## Хранилище
`~/.local/share/hmg/stores/default/` — Данные не покидают машину (Community/Developer Local).

## Архитектура выпусков
Единый бинарный файл. Выпуск определяется лицензионным ключом:
- Без ключа → Community (50K атомов, 5 агентов)
- `<key>` → Developer (безлимит, One-Shot Recall)
- `<key>` → Enterprise (SSO, RBAC, мульти-тенант)

## Границы безопасности
Community: нуль исходящих сетевых соединений. Привязан к localhost. Права доступа только пользователя.
