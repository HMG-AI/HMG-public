# Классификация публикации ADR

Согласно ADR 2026-05-24 (v2) §Граница документации, все ADR должны быть классифицированы перед публикацией.

**Дата:** 2026-05-25

## Резюме
| Классификация | Количество |
|---|---|
| **Public** | 2 |
| **Sanitize** | 6 |
| **Private** | 11 |

## Требуемые действия
1. 2 ADR Public могут быть немедленно скопированы.
2. 6 ADR Sanitize требуют ручной проверки.
3. 11 ADR Private остаются в приватном монорепо.
