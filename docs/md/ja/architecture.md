## コンポーネント

### インターフェース

HMG は4つのアクセス面を提供します：

| インターフェース | プロトコル | 使用ケース |
|---|---|---|
| **MCP** | Model Context Protocol | エージェント統合（主要） |
| **HTTP API** | REST + JSON | SDK 統合、カスタムツール |
| **CLI** | ターミナル（`hmg` コマンド） | 管理、デバッグ、スクリプト |
| **TUI** | インタラクティブターミナル（`hmg tui`） | ビジュアル閲覧と管理 |

4つの面は同じ機能を公開します — 記憶の保存、リコール、修正、ガバナンス。

### メモリエンジン

HMG のコア。以下を管理します：

- **グラフモデル**：型付けされたエッジで接続されたアトム（Supersedes、DerivesFrom、RelatesTo など）
- **インデックス**：キーワード検索、時系列順序、カテゴリ別グループ化、およびセマンティック検索
- **ストレージ**：スナップショット履歴を持つ永続的なローカルストレージ

### エージェント統合

`hmg init --agent <id>` でエージェントを HMG を記憶レイヤーとして使用するように設定します。サポートされているエージェント：

| エージェント | ステータス |
|---|---|
| Cursor | ✅ サポート |
| pi (Codex fork) | ✅ サポート |
| Claude Code | ✅ サポート |
| Codex | ✅ サポート |
| Windsurf | ✅ サポート |
| Aider | ✅ サポート |
| Continue | ✅ サポート |

## データフロー

### 記憶

```
エージェント → "これを覚えて：..." → HMG
  → 入力を検証
  → 型付き記憶アトムを作成
  → スコープとコンテキストを付与
  → 検索用インデックスを構築（キーワード + 時系列 + カテゴリ）
  → ローカルストレージに永続化
  → アトム ID + 確認を返す
```


### リコール

```
エージェント → "データベースは何を選んだ？" → HMG
  → クエリの意図を解析
  → インデックスから候補を検索
  → 関連性、確実性、鮮度でランク付け
  → スコープとガバナンス状態でフィルタ
  → グラフトラバーサルで関連アトムを投影
  → Agent Tool Output Contract に従ってフォーマット
  → 構造化された結果 + 診断情報を返す
```


### 修正

```
エージェント → "それは違う、実際は ..." → HMG
  → 修正アトムを作成（アクション + 理由）
  → Supersedes エッジでリンク
  → 元のアトムの極性/認識状態を更新
  → 修正履歴を永続化
  → 修正確認を返す
```


### ガバナンス

```
管理者 → "この機密記憶を封印して" → HMG
  → ガバナンスアクションを検証
  → 露出状態を遷移（visible → sealed）
  → オプションで安全な教訓アトムを派生
  → ガバナンス記録を永続化
  → 元のコンテンツは取得不可に（sealed）
```


## ストレージ

HMG は実行マシンのローカルにすべてのデータを保存します：

```
~/.local/share/hmg/
  stores/
    default/           ← デフォルト記憶ストア
      graph/           ← アトムとエッジデータ
      indexes/         ← 検索インデックス
      snapshots/       ← 修正/ガバナンス履歴
```

Community および Developer Local エディションでは、データがマシンから出ることはありません。

## エディションアーキテクチャ

HMG はすべてのエディションのコードを含む単一バイナリです。アクティブなエディションは起動時に決定されます：

```
HMG バイナリ
  │
  ├── ライセンスキーなし → Community Edition
  │     └── One-Shot Recall、50K アトム、5 エージェント、数量制限
  │
  ├── HMG_LICENSE_KEY=<your-key> → Developer Edition
  │     └── One-Shot Recall、自動統合、ドメインパック、無制限
  │
  ├── HMG_LICENSE_KEY=<your-key> → Enterprise Edition
  │     └── 全機能、SSO、RBAC、マルチテナント、監査
  │
  └── HMG_CLOUD_TOKEN → クラウド接続
        └── クラウド認証で Developer または Enterprise
```

つまり：
- 個別のバイナリを維持する必要がない
- アップグレードは即座：`export HMG_LICENSE_KEY=...` して再起動
- Community ユーザーは Enterprise と同じバイナリ品質を利用可能

## セキュリティ境界

```
┌─────────────────────────────────┐
│         エージェントプロセス       │
│   （ユーザー権限で実行）            │
└──────────────┬──────────────────┘
               │ MCP / HTTP (localhost)
               ▼
┌─────────────────────────────────┐
│         HMG プロセス              │
│   （localhost:3000 にバインド）    │
│                                  │
│   記憶データ：ユーザーのみアクセス可能│
│   外向き接続なし (CE)              │
│   テレメトリなし (CE)             │
└─────────────────────────────────┘
```

- Community Edition は**ゼロの外向きネットワーク接続**
- HMG はデフォルトで `localhost` にバインド — ネットワークに露出しない
- ストレージファイルはユーザーのみ権限を使用
