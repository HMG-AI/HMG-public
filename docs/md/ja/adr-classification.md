# ADR 公開分類

ADR 2026-05-24 (v2) §ドキュメント境界に従い、すべての ADR は公開リリース前に分類される必要があります。このドキュメントは各 ADR の分類を記録します。

**分類日：** 2026-05-25

## 分類凡例

| マーク | 意味 |
|---|---|
| **Public** | そのまま公開可能 |
| **Sanitize** | 内部アルゴリズム/フックの詳細を削除後に公開可能 |
| **Private** | プライベートモノレポに保持 — プロプライエタリな内部実装を含む |

## 必要なアクション

1. 2つの Public ADR は即座に `export/docs/adr/` にコピー可能。
2. 6つの Sanitize ADR は公開前にプロプライエタリ部分を削除する手動レビューが必要。
3. 11の Private ADR は無期限でプライベートモノレポに保持。
