# ADR 공개 분류

ADR 2026-05-24 (v2) §문서 경계에 따라, 모든 ADR은 공개 릴리스 전에 분류되어야 합니다. 이 문서는 각 ADR의 분류를 기록합니다.

**분류 날짜：** 2026-05-25

## 분류 범례

| 마크 | 의미 |
|---|---|
| **Public** | 그대로 공개 가능 |
| **Sanitize** | 내부 알고리즘/훅 세부사항 제거 후 공개 가능 |
| **Private** | 프라이빗 모노레포에 유지 — 독점적 내부 구현 포함 |

## 필요한 액션

1. 2개의 Public ADR은 즉시 `export/docs/adr/`에 복사 가능.
2. 6개의 Sanitize ADR은 공개 전 독점적 부분 제거를 위한 수동 리뷰 필요.
3. 11개의 Private ADR은 무기한 프라이빗 모노레포에 유지.
