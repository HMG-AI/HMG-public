# HMG 상표 정책

**최종 업데이트：** 2026년 5월 25일

## HMG 상표

다음은 武汉凡尘合创科技有限公司의 상표입니다：

- **HMG**
- **Holographic Memory Graph**
- **HMG Cloud**、**HMG Developer**、**HMG Enterprise**
- **HMG Compatible**、**HMG Certified**
- **One-Shot Recall**
- HMG 로고

*（참고：상표 등록이 진행 중입니다. 등록 완료 후 이 목록이 업데이트됩니다.）*

## 사용 가이드라인

### 가능한 사용

- 일반 텍스트에서 "HMG"를 사용하여 HMG 제품이나 프로토콜을 참조
- 제품이 "HMG Protocol을 구현함" 또는 "HMG와 호환됨"을 명시
- 문서, 블로그 게시물 또는 교육 자료에서 HMG 이름을 사용하여 제품을 설명하거나 리뷰
- 공식 HMG 웹사이트 및 저장소에 링크

### 금지 사항

- 서면 허가 없이 제품명, 도메인명 또는 로고에 HMG 상표 사용
- 승인 없이 공식 보증, 인증 또는 제휴를 암시
- HMG 상표 또는 로고 수정
- HMG 상표를 사용하여 제품 출처에 대해 사용자를 오도함

### "HMG Compatible"

서드파티 구현은 HMG 적합성 테스트 스위트（`hmg-certification`）를 통과한 경우에만 **"HMG Compatible"**을 주장할 수 있습니다.

### "HMG Certified"

공식 HMG 인증 프로세스를 통과한 구현만 **"HMG Certified"** 마크를 사용할 수 있습니다. 인증 요구사항：

1. 모든 적합성 테스트 통과
2. HMG 프로토콜 사양 준수
3. 거버넌스 및 감사 추적 검증
4. HMG AI의 검토 및 승인
