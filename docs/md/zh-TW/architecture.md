## 元件

### 介面

HMG 提供四個存取面：

| 介面 | 協定 | 使用場景 |
|---|---|---|
| **MCP** | Model Context Protocol | Agent 整合（主要方式） |
| **HTTP API** | REST + JSON | SDK 整合、自訂工具 |
| **CLI** | 終端（`hmg` 命令） | 管理、除錯、腳本 |
| **TUI** | 互動式終端（`hmg tui`） | 視覺化瀏覽和管理 |

四個面暴露相同的能力——儲存記憶、召回、糾錯和治理。

### 記憶引擎

HMG 的核心。管理以下內容：

- **圖譜模型**：透過型別化邊連接的原子（Supersedes、DerivesFrom、RelatesTo 等）
- **索引**：關鍵詞搜尋、時序排序、分類分組，以及（Developer+ 版本中的）語義搜尋
- **儲存**：帶快照歷史的持久化本地儲存

### Agent 整合

`hmg init --agent <id>` 設定 agent 使用 HMG 作為其記憶層。支援的 agent：

| Agent | 狀態 |
|---|---|
| Cursor | ✅ 已支援 |
| pi (Codex fork) | ✅ 已支援 |

| Codex | ✅ 已支援 |
| Windsurf | ✅ 已支援 |
| Aider | ✅ 已支援 |
| Continue | ✅ 已支援 |

## 資料流

### 記憶

```
Agent → "記住這個：..." → HMG
  → 驗證輸入
  → 建立型別化記憶原子
  → 附加範圍和上下文
  → 建立檢索索引（關鍵詞 + 時序 + 分類）
  → 持久化到本地儲存
  → 返回原子 ID + 確認
```


### 召回

```
Agent → "資料庫用什麼？" → HMG
  → 解析查詢意圖
  → 從索引檢索候選
  → 按相關性、確定性、時效排序
  → 按範圍和治理狀態過濾
  → 透過圖遍歷投影相關原子
  → 按 Agent Tool Output Contract 格式化輸出
  → 返回結構化結果 + 診斷資訊
```


### 糾錯

```
Agent → "那個不對，實際上是 ..." → HMG
  → 建立糾錯原子（動作 + 原因）
  → 透過 Supersedes 邊連接
  → 更新原始原子的極性/認識狀態
  → 持久化糾錯歷史
  → 返回糾錯確認
```


### 治理

```
管理員 → "密封這條敏感記憶" → HMG
  → 驗證治理動作
  → 轉換暴露狀態（可見 → sealed）
  → 可選擇衍生安全教訓原子
  → 持久化治理記錄
  → 原始內容變為不可檢索（sealed）
```


## 儲存

HMG 將所有資料儲存在執行機器的本地：

```
~/.local/share/hmg/
  stores/
    default/           ← 預設記憶儲存
      graph/           ← 原子和邊資料
      indexes/         ← 搜尋索引
      snapshots/       ← 糾錯/治理歷史
```

Community 和 Developer Local 版本中，資料不會離開本機。

## Edition 架構

HMG 是一個包含所有版本程式碼的單一二進位。活動版本在啟動時決定：

```
HMG 二進位
  │
  ├── 無 license key → Community Edition
  │     └── 關鍵詞搜尋，50K 原子，5 個 agent，基礎功能
  │
  ├── HMG_LICENSE_KEY=<your-key> → Developer Edition
  │     └── One-Shot Recall，自動整理，Domain Packs，無限量
  │
  ├── HMG_LICENSE_KEY=<your-key> → Enterprise Edition
  │     └── 全部功能，SSO，RBAC，多租戶，審計
  │
  └── HMG_CLOUD_TOKEN → 雲連接
        └── 透過雲端認證獲得 Developer 或 Enterprise
```

這表示：
- 不需要維護多個二進位
- 升級即時生效：`export HMG_LICENSE_KEY=...` 然後重啟
- Community 使用者獲得與 Enterprise 相同的二進位品質

## 安全邊界

```
┌─────────────────────────────────┐
│         Agent 行程               │
│   （以使用者權限執行）             │
└──────────────┬──────────────────┘
               │ MCP / HTTP (localhost)
               ▼
┌─────────────────────────────────┐
│         HMG 行程                 │
│   （綁定到 localhost:7654）       │
│                                  │
│   記憶資料：僅使用者可存取         │
│   無對外連線 (CE)                 │
│   無遙測 (CE)                    │
└─────────────────────────────────┘
```

- Community Edition **零對外網路連線**
- HMG 預設綁定到 `localhost`——不暴露到網路
- 儲存檔案使用僅使用者權限
